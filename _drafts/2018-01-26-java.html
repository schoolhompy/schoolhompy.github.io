7.static 변수를 임포트했을떄, Main 에 같은 이름의 static 변수가 있다면 Main 에 있는것을 사용한다.
package ex7;
public class Sample{
    public final static int VALUE=100;
}

import ...ex8.Sample.VALUE;
public class Main() {
    public final static int VALUE=0;
    void main() {
        VALUE <- 0 이됨
    }
}
-또한 같은 이름의 변수가 다른 타입으로 중복선언되도 에러.
