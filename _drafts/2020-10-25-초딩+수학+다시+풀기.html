1.할인

1할은 10%, 2할5분 은 25%
어떤물건의 2할5분 이라면 640원 x 0.25 = 160원 를 하면됨.
반대로 2할5분을 할인받아 160원이라면 160 / 0.25 -> 160 x (100/25) = 640 이됨.
*** 나누기를 계산할때 분수와 분수 나누는 것은 복잡하니까, 두번째 식의 분모와 분자를 바꾸면 곱하기로 바꿀수 있음.
예) (1/3)   / (5/7) 은 복잡하지만, 1/3 x 7/5 는 위에것은 위에것만, 아래것은 아래것만 곱하면 되니까 1x7  / 3x5 = 7 / 15 이 된다. 이렇게 하면 분수 나눗셈이 조금 쉬움(초딩 4,5학년 자료참고.컥)

비율관련

포인트:
큰개체를 기준으로 포함된애들의 비율을 곱하면 작은객체의 크기가 나온다. 작은객체의 크기를 안다면 반대로 해준다.

1. 작업속도가 다른 3대의 기계 P , Q , R 있다. Q 가 P보다 작업량이 1할적고, R은 Q보다 2할많다.
R은 P보다 몇 % 작업량일까?
공식을 쓰면 Q = P x 0.1 , R = Q x 1.2 이다.
R을 알고 싶고 작업의 기준이 P이니까 100을  P에 값을 넣어보면,
Q = 100 x 0.1 = 90
R = 90 x 1.2 = 108 이다.
 

2. 학교의 부활동중의 학생수는 전체학생의 70%이다. 부활동을 하는 학생의 남녀비율은 7:5 이다.

(1) 부활동을 하는 남학생은 392명이었다. 학교의  전체 남학생은 몇명일까?

전체학생에서 부활동을 하는 애들의 비율은 0.7 이고 이중 남녀 비율은 7:5 인데 부활동하는 남학생은 392 명이었다.  큰객체는 모르지만 작은 객체와 소속된 비율을 알고 있기에 큰 객체만 X로 놓자.

X x 0.7 x (7/12) = 392 -> X x (4.9/12) = 392 -> X = 392 x 12 /4.9 -> 80 x 12/1 = 960

(2) 부활동을 하지않는 학생의 남녀비율은 5:7 이었다. 여학생중에서 부활동 하고 있는 학생은 몇 %인가

부활동을 하는 여학생은 960 x 0.7 x ( 5/12) = 280 

부활동을 하지않는 여학생은 960 x 0.3 x ( 7/12) = 168 명이니까

전체 여학생중에서 부활동을 하고 있는 학생은 280 / 448 x100 = 62.5 % 이다

3. 유원지 



할인관련

포인트 :

어떤 상품을 선지급하고 나머지잔액을 할부 납부할때는 잔액부분과 할부개수를 곱함
( 선지급금액을 뺀 나머지 구입금액의 비율 x 잔액에 대한 할부 개월수 

선지급을 1/3 만큼 한 상태에서 잔액을 납무할경우 2/3 x 1/15 = 2/45 를 매회 납부하면됨. 

1. 어떤 기계를 살때 1/5 을 먼저 계산후, 잔액을 7회 분할납부하기로 했다.

(1) 1회마다 분납할 금액은 어느정도일까? 

잔액이니까 1-1/5 = 4/5

7회납부니까 (4/5) /7 공식이 귀찮으니까 4/5 x 1/7 하면 4/35 이다

(2) 7회 분납중 3회째 납부했을때의 누적된 총납부금액은 총상품금액의 어느정도일까

1회때마다 4/35 니까 3회라면 4/35 x 3 하면 12/35 이다. 여기서 1/5 을 먼저 납부했다고 했으니 1/5 + 12/35 (7/35 + 12/35) 하면 19/35 이다. 

2. 게임기를 3회에 할부하기로 하고 처음에 1/6 을 납부했다.

(1) 2회째에 총액의 3/8 을 납부하고, 3회째에 남은 금액 전부 납부할때 3회째에는 총금액의 어느정도를 납부해야 할까.

1회째와 2회째 지불했던 금액을  ¥더하면 1/6 + 3/8 = 26/48 = 13/24

남은 금액은 전체에서 빼야되니까 1-13/24 = 11/24

(2) 2회째에 처음1회때 금액의  2/3 를 지급, 3회째에 남은 금액 전부 납부할때 1회때 납부 금액의 어느정도일까

2회째에는 1/6 x 2/3 = 2/18 니까 결국 5/18 를 납부한거임

3회째에는 (13/18) 인데  1회때에 대한 비율이니까 (13/18 )  / (3/18) 를 쉽게 13/18 x 6/1 하면 , 13/3 

3. 티비구입할떄, 잔액을 8회 분할납부 하되, 잔액에 대한 분할납부수수료로 잔액금액의 1/5 이 추가됨. 

(1) 구매시 선지급금액을 1/5 납부했을때 , 남은 잔액의 1회마다 납부해야할 금액

4/5 를  분할납부할때 총잔액의 1/5 을 추가수수료로 내야되니까 

총잔액에서 1/5 비율만큼 늘려야 해서   (4/5) x (6/5) = 24/25

이걸 다시 매회 얼마만큼 내야되는지 알아야 하니까 (24/25) / (8) 를 쉽게 24/25 x 1/8 하면 3/25 

(2) 분할납부의 금액을 1/10 로 하기 위해서 , 선지급금액을 얼마로 해야좋을까

분할납부금액 1/10 을 8번에 내면 잔액전부 지급되는거니까 총금액(분할수수료 포함된금액) 에서 내야할 잔액 부분은 ,  1/10  x 8 = 4/5 정도

근데 선지급금은 분할 수수료만큼 추가된 거니까 수수료비율(잔액의 1/5) 만큼을 계산해서 수수료가 없을경우에 납부금액을 1/0으로 하기위한 계산을 해야하니까

(4/5) / (6/5) 쉽게해서 4/5 x 5/6  = 1/3
